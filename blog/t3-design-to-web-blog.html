Technical Blog Part 3
(Udacity Reflection)

  • Responsive site;

    • A responsive website is a single website that adapts to any screen size so it's as easy to use on mobile as it is on tablet and desktop. This is importnat because you don't know what kind of device somebody will be viewing your webpage on and you want it to look great !

  • Mobile first site;

    • Mobile first is designing the layout, navigation and content of a website BEFORE designing the desktop versions for larger screens.

  • Frameworks;

    • These are pre-prepared, preset style sheets which contain already established grids and layout. These are great for smaller simple projects but this can restrict your freedom and own idea's when creating a webpage.

  • Wireframes;

    • An image or set of images which displays the functional elements of a website or page, typically used for planning a site's structure and functionality. I strugled with getting the correct spacing with the margins and padding.
